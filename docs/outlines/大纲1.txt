## 2.4 图像分割

在人脸属性分析任务中，尽管前述对齐、滤波与对比度增强步骤已在一定程度上提升了图像质量，但原始图像中仍常包含复杂背景、局部遮挡与非目标区域纹理（如衣物、头发外侧、背景物体等）。这些非相关信息会在特征提取与分类阶段引入额外的统计波动，降低人脸区域的**信噪比（Signal-to-Noise Ratio, SNR）**，并造成不必要的**计算冗余**（例如卷积特征图对背景纹理的响应）。因此，本文在预处理流程中引入图像分割模块，旨在将模型关注范围聚焦于人脸主体区域（Region of Interest, ROI），从而削弱背景干扰、提升面部结构的可辨识性，并为后续特征提取与属性判别提供更加稳定的输入表示。

### 2.4.1 基于 Otsu 阈值的分割与形态学优化

为获得无需人工调参的二值化分割结果，本文采用大津法（Otsu’s method）作为全局阈值自动选取策略。Otsu 方法属于非参数化（non-parametric）阈值分割方法，其核心思想是在灰度直方图意义下寻找一个阈值 (t)，使得前景与背景两类的**类间方差**最大，从而获得全局意义下的最优分割。设灰度级为 (0,1,\dots,L-1)，以阈值 (t) 将像素划分为两类：(C_0={0,\dots,t}) 与 (C_1={t+1,\dots,L-1})。两类的概率（权重）分别为
[
\omega_0(t)=\sum_{i=0}^{t}p(i),\quad \omega_1(t)=\sum_{i=t+1}^{L-1}p(i)=1-\omega_0(t),
]
其中 (p(i)) 表示灰度级 (i) 的归一化直方图概率。对应的类内均值为
[
\mu_0(t)=\frac{1}{\omega_0(t)}\sum_{i=0}^{t} i,p(i),\quad
\mu_1(t)=\frac{1}{\omega_1(t)}\sum_{i=t+1}^{L-1} i,p(i),
]
全局均值为 (\mu_T=\sum_{i=0}^{L-1} i,p(i))。类间方差定义为
[
\sigma_B^2(t)=\omega_0(t)\big(\mu_0(t)-\mu_T\big)^2+\omega_1(t)\big(\mu_1(t)-\mu_T\big)^2
=\omega_0(t),\omega_1(t)\big(\mu_0(t)-\mu_1(t)\big)^2.
]
Otsu 法通过遍历 (t) 并选择使 (\sigma_B^2(t)) 最大的阈值 (t^*)，即
[
t^*=\arg\max_t \sigma_B^2(t),
]
从而实现对不同对比度样本的自适应阈值选取。该策略的优势在于无需依赖先验参数，且当人脸区域与背景在亮度分布上存在可分性时，能够获得稳定的二值分割结果。

然而，在人脸图像场景中，光照阴影、头发边缘、眼镜反光等因素常导致阈值分割结果出现孤立噪点与内部孔洞，直接影响掩膜的连通性与结构完整性。为此，本文在 Otsu 二值化后引入形态学后处理（morphological post-processing）以增强分割鲁棒性。具体而言，采用**开运算（Opening）**与**闭运算（Closing）**作为互补操作：开运算可表示为
[
A\circ B=(A\ominus B)\oplus B,
]
其通过“先腐蚀后膨胀”抑制小尺度孤立噪点，适用于消除背景杂波或阈值误分割产生的离散前景点；闭运算可表示为
[
A\bullet B=(A\oplus B)\ominus B,
]
其通过“先膨胀后腐蚀”填补前景内部的小孔洞并连接断裂区域，尤其能缓解眼睛阴影、鼻翼暗部或光照不均导致的掩膜破碎问题。通过上述处理，可获得结构更完整、连通性更强的二值人脸掩膜，为后续 ROI 提取提供可靠基础。

### 2.4.2 掩膜应用与结果展示

在获得二值掩膜后，本文采用位运算（bitwise operation）将掩膜应用于对齐人脸图像，以显式提取感兴趣区域。设输入图像为 (I)，二值掩膜为 (M\in{0,1})，则 ROI 提取可表示为逐像素乘法或按位与操作：
[
I_{\text{ROI}} = I \odot M,
]
其中 (M=1) 的区域保留原始像素，(M=0) 的区域置零或替换为统一背景值，以减少背景纹理对后续特征响应的干扰。该处理等价于在输入域实施“硬注意力（hard attention）”，能够提升网络对面部核心区域的聚焦程度，并降低对背景冗余信息的计算开销。

从 **Figure 10** 可见，Otsu+形态学策略能够在多数样本上产生较为完整的人脸掩膜，背景被有效抑制，面部主体结构得到保留。值得注意的是，在复杂边缘区域（例如眼镜框边缘、发际线与头发过渡处），由于局部对比度变化与细碎纹理的存在，掩膜可能出现轻微的边界锯齿或局部欠分割现象；但总体而言，该误差主要集中在边缘窄带区域，对人脸主体（五官与面部轮廓）覆盖影响有限。由于后续属性判别（如性别、眼镜）往往依赖面部整体结构与眼部周围区域信息，适度的边缘误差通常不会显著降低识别性能，反而在整体上通过背景抑制提升了输入一致性与鲁棒性。

### 2.4.3 K-means 聚类分割（对比实验）

为验证分割策略选择的合理性，本文进一步引入 K-means 聚类作为对比实验。K-means 属于无监督聚类方法，其目标是在像素特征空间中将样本划分为 (K) 个簇（本文取 (K=3)），以最小化簇内平方误差：
[
\min_{{\mu_k}*{k=1}^{K}} \sum*{k=1}^{K}\sum_{x_i\in C_k}|x_i-\mu_k|^2,
]
其中 (x_i) 表示像素的颜色向量（可取 RGB/YCrCb 等空间），(\mu_k) 为第 (k) 个簇中心。该方法能够在颜色分布层面将图像粗略划分为不同区域（如肤色、头发、背景），并为无标注场景提供一种直观的分割途径。

[Insert Figure 11 here]
**Figure 11.** 基于 K-means（(K=3)）的无监督颜色聚类分割示例。（a）对齐后人脸图像；（b）K-means 聚类得到的颜色分割结果（不同颜色表示不同簇）。

然而，K-means 的关键局限在于其主要依据像素值相似性进行聚类，而未显式建模空间连续性（spatial continuity）与形状先验。在人脸图像中，同一面部区域常因阴影、反光与肤色渐变产生明显的颜色差异，导致脸部被错误地划分为多个色块（例如将阴影区与正常肤色区分到不同簇），同时头发与背景也可能因颜色接近而产生混淆。由 **Figure 11** 可见，K-means 分割结果往往呈现块状碎片化，面部区域缺乏整体连通性，难以形成“结构完整的人脸掩膜”。这种碎片化分割在后续 ROI 提取时会造成面部信息缺失或引入大量不连续边界，不利于稳定的特征统计与属性判别。

综合对比 **Figure 10** 与 **Figure 11** 可知：Otsu 阈值分割通过最大化类间方差获得全局最优阈值，并在形态学处理的辅助下显著提升掩膜的连通性与结构完整性，对光照变化与局部阴影具有更好的工程鲁棒性；而 K-means 由于忽略空间约束，虽能实现颜色层面的粗分，但易产生面部碎片化与区域混淆。基于上述实验现象与下游任务需求，本文在整体流程中优先采用 **Otsu + 形态学优化** 作为人脸区域分割方案，以获得更稳定、可解释且适用于属性分析的 ROI 表示。
